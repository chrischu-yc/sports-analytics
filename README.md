## Statsbomb Open Data Match Explorer
A website that showcases all matches in the Statsbomb open data, and contains many useful visuals including passing maps, team passing networks, player performance cards and a lot more.   
Right now this is version one, with already quite a lot of features I think. The next thing I'm going to add is probably goalkeeper-specific performance cards (when I have time :)

## F1 Race Trace App
An interactive Streamlit app for exploring Formula 1 race data with FastF1. It lets you pick a season and race, load the session once, and then inspect multiple views of the same event without rerunning the data load each time.

The app includes a race trace overview with driver pace offsets, pit stop markers, and safety car shading, plus a team pace comparison chart. A second tab adds team-specific visuals for the selected constructor, including fastest-lap telemetry, track maps, and lap-time scatter plots.

### Features
- Year and race selection with dependent race options.
- Load-once race sessions with persistent selections in Streamlit.
- Race trace overview with filtered driver and lap-range views.
- Team pace comparison boxplots and tyre strategies comparison.
- Team-specific telemetry, track map, and lap-time charts.

The app is powered by the FastF1 python package and is intended for educational and entertainment use.
